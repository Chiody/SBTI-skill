## 本仓库实际包含什么

> **📌 与纯「Skill 单目录」型项目不同**：本开源仓是 **网页 + 微信小程序 + Cursor Agent Skill** 一体交付，方便你本地玩、改题、上架小程序，并让 Cursor 按约定改代码不翻车。

| 模块 | 路径 / 说明 |
|------|-------------|
| 网页单页 | 根目录 `index.html` + `image/` |
| 微信小程序 | `miniprogram/`（海报在分包 `packageImages/`） |
| Cursor Skill | `.cursor/skills/sbti-ti-stack/SKILL.md` 与 `reference.md` |
| 数据同步脚本 | `scripts/build-sbti-data.js`、`scripts/sync-mp-images.sh` |
| 冒烟校验 | `scripts/verify-sbti-engine.js` |

## 安装

### 克隆仓库

**仓库地址**：`https://github.com/Chiody/SBTI-skill`

```bash
git clone https://github.com/Chiody/SBTI-skill.git
cd SBTI-skill
```

### 网页本地预览

```bash
chmod +x ./start-local.sh
./start-local.sh
```

浏览器打开终端提示的地址（默认 `http://127.0.0.1:8765/`）。

### 微信小程序

使用 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html) 导入**仓库根目录**（含 `project.config.json`），填写你的 `appid` 后编译预览。

### Cursor Agent Skill

Skill 已随仓库位于：

`.cursor/skills/sbti-ti-stack/`

在 Cursor 中打开本仓库即可被识别；若要在其他项目引用，可将该目录复制到你的项目 `.cursor/skills/` 下。

### Claude Code（可选）

若使用 Claude Code，可将 Skill 目录挂到其技能路径（以你本机 Claude 配置为准），例如：

```bash
# 在目标 git 仓库根目录执行（示例：复制 Skill 子目录）
mkdir -p .claude/skills
cp -R /path/to/SBTI-skill/.cursor/skills/sbti-ti-stack .claude/skills/sbti-ti-stack
```

> 下列「斜杠命令」为**交互范式参考**；本仓库主交付为可运行网页/小程序 + Skill 文档， slash 命令需在对应 AI 工具内自行接入实现。

## 使用

### 人类玩家

- **网页**：`./start-local.sh` 后按页面完成全部题目并查看结果。  
- **小程序**：开发者工具或真机中完整走答题 → 结果页（含十五维、海报、饮酒分支逻辑）。

### Cursor / Agent 维护者

阅读 `.cursor/skills/sbti-ti-stack/SKILL.md`，改题时以 `index.html` 为单一事实来源，再运行：

```bash
node scripts/build-sbti-data.js
./scripts/sync-mp-images.sh
```

### 若在 Claude Code 中实现了 `/create-sbti` 类产品化封装（自建）

可按你的产品流程输入 `/create-sbti`，按提示完成测试并生成结果（需自行实现 prompts 与命令绑定）。**本仓库默认不内置**下列命令，仅供你扩展时对照：

| 命令 | 说明 |
|------|------|
| `/list-sbti` | 列出已生成过的 SBTI 人格（自建） |
| `/sbti-{type}` | 以对应人格语气交互（自建） |
| `/sbti-work` | 仅工作状态（自建） |
| `/sbti-persona` | 仅人格语气（自建） |
| `/delete-sbti {type}` | 删除对应人格（自建） |

## 功能特性

- 🛡 **本地隐私优先**：测试与计分可在本地完成，本仓库默认 **零外链**（无统计脚本、无远程图床回退）。  
- 🎭 **全人格支持**：数据侧覆盖完整类型库与匹配模式（与网页/小程序同源）。  
- 🎲 **洗牌与分支**：支持题序随机、饮酒相关隐藏题等逻辑（与原版玩法对齐）。  
- 📤 **结果展示**：网页与小程序结果页含海报与长文案解读。  
- 📦 **可维护**：`scripts/` 同步双端数据；`verify-sbti-engine.js` 冒烟校验。  
- 🔄 **可扩展**：可自行加分享图生成、版本存档、增量合并等（欢迎 PR）。
